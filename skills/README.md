# OVERVIEW
This repository contains some reusable ai resources like prompt, skills, agent...etc


### windows utility
powershell script(`make-symbolic-links.ps1`) transfer resources to `~/User/<username>/.codex/skills/` and `~/User/<username>/.claude/skills/` as symbolic link.
```powershell
path/to/AIResources$ pwsh make-symbolic-links.ps1
```
